# Blog Fullstack - React Node MongoDB

Projet de blog complet avec authentification.

## Stack Technique
- **Frontend:** React, React Router, Axios
- **Backend:** Node.js, Express, MongoDB
- **Auth:** JWT, Bcrypt

## Installation

### Backend
```bash
cd backend
npm install
# Configurer .env avec tes variables
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
